## Tentang Aplikasi

SI-PANDU ZI adalah aplikasi berbasis web yang digunakan oleh Kelompok Kerja Pengawas Madrasah (Pokjawas) Kementerian Agama Kabupaten Jember untuk:

- Mengelola data pengawas dan madrasah binaan
- Mencatat kegiatan pendampingan Zona Integritas (ZI)
- Memantau progress checklist eviden ZI
- Menyimpan dokumen bukti (eviden)
- Memonitor capaian ZI setiap madrasah
- Menyediakan kanal survei kepuasan dan pengaduan publik

---

## Cara Mengakses Aplikasi

- Buka browser (Chrome, Firefox, Edge, atau Safari)
- Ketik alamat: **https://subariyanto.github.io/si-pandu-zi-madrasah/**
- Halaman login akan muncul

## Login

- Masukkan **Username** atau **NIP** pada kolom yang tersedia
- Masukkan **Password**
- Klik tombol **Login**
- Akun dibuat oleh Admin (pengguna tidak bisa mendaftar sendiri)
- Jika lupa password, hubungi Admin untuk direset

## Role (Peran) Pengguna

Aplikasi ini memiliki 4 peran pengguna dengan hak akses berbeda:

### 1. Admin

- Mengelola semua data (pengawas, madrasah, pendampingan, dll.)
- Membuat akun login untuk pengawas dan kepala madrasah
- Mengatur pengaturan sistem
- Import dan export data
- Melihat semua laporan dan rekapitulasi

### 2. Ketua Pokjawas

- Melihat semua data pengawas dan madrasah
- Monitoring progress pendampingan ZI
- Melihat dashboard, laporan, dan rekapitulasi
- Tidak dapat mengedit data (hanya melihat/monitoring)

### 3. Pengawas

- Mengelola data madrasah binaannya
- Mencatat kegiatan pendampingan ZI
- Mengisi checklist eviden ZI
- Melihat kartu kendali madrasah binaannya
- Melakukan konsultasi via Klinik ZI

### 4. Madrasah (Kepala Madrasah/Operator)

- Melihat data madrasahnya sendiri
- Mengupload eviden/dokumen bukti
- Melihat progress checklist dan kartu kendali madrasahnya

---

## Alur Kerja Aplikasi

Berikut urutan kerja penggunaan aplikasi dari awal:

1. **Admin menginput data pengawas**
   - Bisa input manual satu per satu
   - Bisa import dari file Excel (download template dulu)

2. **Admin membuatkan akun login untuk pengawas**
   - Username/NIP + Password
   - Informasikan ke pengawas yang bersangkutan

3. **Pengawas login dan menambah data madrasah binaan**
   - Pengawas masuk ke menu Data Madrasah
   - Tambahkan madrasah yang menjadi binaannya

4. **Admin membuatkan akun login untuk kepala madrasah**
   - Username/NIP + Password
   - Informasikan ke kepala madrasah yang bersangkutan

5. **Pengawas melakukan pendampingan**
   - Setelah kunjungan, catat di menu Pendampingan ZI
   - Isi tanggal, kegiatan, hasil, dan tindak lanjut

6. **Pengawas mengisi checklist eviden**
   - Centang item-item eviden yang sudah terpenuhi per area perubahan

7. **Madrasah mengupload eviden/dokumen**
   - Kepala madrasah/operator login
   - Upload file bukti dokumen sesuai area perubahan

8. **Monitoring dan evaluasi**
   - Pantau progress via Dashboard, Kartu Kendali, dan Rekapitulasi Capaian
   - Cetak laporan triwulan sesuai kebutuhan

## Panduan Per Menu/Fitur

### Dashboard

- Halaman utama setelah login
- Menampilkan statistik ringkasan:
  - Jumlah pengawas
  - Jumlah madrasah
  - Jumlah pendampingan
  - Progress capaian ZI
- Menampilkan grafik visual untuk memudahkan monitoring
- Semua role bisa melihat dashboard (sesuai hak aksesnya)

### Data Pengawas

**Fungsi:** Mengelola data pengawas madrasah

**Fitur yang tersedia:**
- **Tambah data** — klik tombol Tambah, isi formulir, lalu Simpan
- **Edit data** — klik ikon Edit pada baris data yang ingin diubah
- **Hapus data** — klik ikon Hapus (akan ada konfirmasi)
- **Import Excel** — upload file Excel sesuai template
- **Download Template** — unduh file template Excel untuk import
- **Export PDF** — cetak/unduh data dalam format PDF
- **Export Excel** — unduh data dalam format Excel

**Langkah import data dari Excel:**
- Klik tombol Download Template
- Isi data pengawas di file template tersebut
- Simpan file Excel
- Klik tombol Import Excel
- Pilih file yang sudah diisi
- Data akan otomatis masuk ke sistem

### Data Madrasah

**Fungsi:** Mengelola data madrasah binaan

**Fitur yang tersedia:**
- Tambah, edit, dan hapus data madrasah
- Filter berdasarkan jenjang (MI/MTs/MA)
- Filter berdasarkan kecamatan
- Data madrasah terhubung dengan pengawas pembinanya

**Catatan:**
- Pengawas hanya bisa melihat dan mengelola madrasah binaannya sendiri
- Admin bisa melihat dan mengelola semua data madrasah

### Pendampingan ZI

**Fungsi:** Mencatat kegiatan kunjungan pendampingan Zona Integritas

**Cara menggunakan:**
- Klik menu Pendampingan ZI
- Klik tombol Tambah Pendampingan
- Pilih madrasah yang didampingi
- Isi informasi:
  - Tanggal kunjungan
  - Kegiatan yang dilakukan
  - Hasil pendampingan
  - Tindak lanjut
- Klik Simpan

**Catatan:**
- Setiap kunjungan dicatat terpisah
- Riwayat pendampingan bisa dilihat per madrasah

### Checklist Eviden ZI

**Fungsi:** Mencentang/checklist kelengkapan eviden per area perubahan ZI

**Area perubahan yang dichecklist:**
- Manajemen Perubahan
- Penataan Tatalaksana
- Penataan Sistem Manajemen SDM
- Penguatan Akuntabilitas
- Penguatan Pengawasan
- Peningkatan Kualitas Pelayanan Publik

**Cara menggunakan:**
- Pilih madrasah
- Pilih area perubahan
- Centang item eviden yang sudah terpenuhi
- Simpan perubahan
- Progress akan otomatis terupdate di dashboard dan kartu kendali

### Upload Eviden

**Fungsi:** Mengupload dokumen bukti/eviden ZI

**Cara menggunakan:**
- Login sebagai Madrasah (Kepala/Operator) atau Pengawas
- Pilih menu Upload Eviden
- Pilih area perubahan
- Pilih file dokumen dari komputer/HP
- Klik Upload
- File akan tersimpan di sistem

**Format file yang didukung:** PDF, gambar (JPG/PNG), dokumen (DOC/DOCX)

**Tips:**
- Beri nama file yang jelas (contoh: SK_Tim_ZI_2026.pdf)
- Pastikan file tidak terlalu besar (maksimal sesuai ketentuan sistem)
- Upload eviden sesuai area perubahan yang benar

### Kartu Kendali

**Fungsi:** Monitoring progress pendampingan dan capaian ZI per madrasah

**Informasi yang ditampilkan:**
- Nama madrasah
- Pengawas pembina
- Progress checklist eviden (persentase)
- Status pendampingan
- Catatan tindak lanjut

**Cara menggunakan:**
- Klik menu Kartu Kendali
- Pilih madrasah (atau lihat semua jika Admin/Ketua Pokjawas)
- Lihat detail progress dan status

### Klinik ZI

**Fungsi:** Fasilitas konsultasi dan bimbingan terkait Zona Integritas

**Cara menggunakan:**
- Klik menu Klinik ZI
- Ajukan pertanyaan atau konsultasi
- Pengawas/Admin dapat memberikan jawaban/bimbingan
- Riwayat konsultasi tersimpan di sistem

### Survei Kepuasan (Publik)

**Fungsi:** Mengumpulkan umpan balik kepuasan dari masyarakat/stakeholder

**Karakteristik:**
- Bisa diakses tanpa login
- Bisa diakses melalui QR Code
- Hasil survei bisa dilihat oleh Admin dan Ketua Pokjawas

**Cara mengakses:**
- Scan QR Code yang disediakan, atau
- Buka link survei langsung dari browser
- Isi formulir survei
- Klik Kirim

### Kanal Pengaduan (Publik)

**Fungsi:** Menerima pengaduan dari masyarakat terkait layanan madrasah

**Karakteristik:**
- Bisa diakses tanpa login
- Bisa diakses melalui QR Code
- Pengaduan masuk akan dilihat oleh Admin

**Cara mengakses:**
- Scan QR Code yang disediakan, atau
- Buka link pengaduan langsung dari browser
- Isi formulir pengaduan (identitas bisa anonim)
- Klik Kirim

### Laporan Triwulan

**Fungsi:** Menyusun dan mencetak laporan pendampingan ZI per triwulan

**Cara menggunakan:**
- Klik menu Laporan Triwulan
- Pilih periode triwulan (Triwulan 1/2/3/4)
- Sistem akan menampilkan ringkasan kegiatan pada periode tersebut
- Klik Cetak PDF atau Export Excel untuk mengunduh

### Rekapitulasi Capaian

**Fungsi:** Menampilkan grafik dan ringkasan capaian ZI per madrasah

**Informasi yang ditampilkan:**
- Grafik progress per madrasah
- Perbandingan capaian antar madrasah
- Persentase kelengkapan eviden per area perubahan

**Cara menggunakan:**
- Klik menu Rekapitulasi Capaian
- Lihat grafik dan data yang ditampilkan
- Bisa difilter per madrasah atau per area perubahan

### Cetak/Export

**Fungsi:** Mencetak atau mengunduh data dalam format PDF dan Excel

**Tersedia di berbagai menu:**
- Data Pengawas → Export PDF / Export Excel
- Data Madrasah → Export PDF / Export Excel
- Laporan Triwulan → Cetak PDF / Export Excel
- Kartu Kendali → Cetak PDF

**Cara menggunakan:**
- Buka menu yang datanya ingin dicetak/diunduh
- Klik tombol Export PDF atau Export Excel
- File akan otomatis terunduh ke komputer/HP Anda

### Pengaturan Akun

**Untuk Admin:**
- Membuat akun baru (untuk pengawas dan kepala madrasah)
- Mengedit akun yang sudah ada
- Mereset password pengguna
- Menonaktifkan akun

**Untuk Semua Pengguna:**
- Mengganti password sendiri

**Cara mengganti password:**
- Klik menu Pengaturan Akun atau ikon profil
- Pilih Ganti Password
- Masukkan password lama
- Masukkan password baru (2 kali untuk konfirmasi)
- Klik Simpan

## Fitur Publik (Tanpa Login)

Dua fitur berikut bisa diakses oleh siapa saja tanpa perlu login:

### Survei Kepuasan

- Akses melalui QR Code atau link langsung
- Isi formulir penilaian kepuasan layanan
- Tidak perlu membuat akun
- Hasil digunakan untuk evaluasi internal

### Kanal Pengaduan

- Akses melalui QR Code atau link langsung
- Sampaikan pengaduan atau masukan
- Bisa anonim (tanpa menyebutkan identitas)
- Pengaduan akan ditindaklanjuti oleh tim

---

## Tips dan Catatan Penting

- **Gunakan browser terbaru** untuk pengalaman terbaik (Chrome, Firefox, Edge)
- **Jangan bagikan password** ke orang lain
- **Logout setelah selesai** menggunakan aplikasi, terutama di komputer bersama
- **Backup data** secara berkala dengan fitur Export Excel
- **Hubungi Admin** jika mengalami kendala teknis atau lupa password
- **Upload eviden secara rutin** agar progress monitoring selalu terupdate
- **Isi pendampingan segera** setelah kunjungan agar data tidak terlewat
- **Manfaatkan fitur filter** untuk mempermudah pencarian data (filter jenjang, kecamatan)
- **Cetak laporan triwulan** sebelum batas waktu pelaporan
